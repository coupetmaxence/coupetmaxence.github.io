I. Installation du bot Telegram

Le bot Telegram a été réalisé avec le langage Python à l'aide de cette bibliothèque et il est inspiré des exemples du github.

Voici un lien vers mon répertoire github personnel contenant le fichier Python (.py) du bot ainsi qu'un fichier README.md expliquant en détail toutes les étapes de l'installation pour que le bot fonctionne.

Vous devez ensuite créer votre propre bot Telegram en parlant avec le BotFather sur Telegram ce qui est expliqué sur ce lien (Pour discuter avec le BotFather il suffit de taper "@BotFather" dans le menu de recherche de contact). Une fois votre bot créé, notez son token (chaîne de caractères assez longue) que vous a renvoyé le BotFather et entrez sa valeur dans le code Python à la ligne 15.

Note : Le script Python doit être en train de s'exécuter pour pouvoir utiliser le bot et le bot est en anglais.

II. Utilisation du bot

Le bot est muni d'une commande /help qui liste toutes les commandes qu'il gère et explique brièvement comment les utiliser.

Il suffit alors d'entrer ces commandes, en insérant bien l'anti-slash avant le nom de la commande et avec les bon paramètres pour que le bot s'exécute.
