Description du projet

Spot&Walk est une application ayant pour but de donnez un autre aspect au tourisme, en permettant à chacun des utlisateurs d'ajouter les liens qu'ils trouvent intéressants, et ainsi partager leurs bonnes adresses à toute la communauté.

Logo de notre application

Ce projet a été réalisé par un groupe de 5 étudiants de l'ESILV, qui ont travaillés sur différentes parties du projet et je me suis occupé de mon côté du dévellopement de l'application sur la plateforme Android.

Fonctionnement de l'application

Nous disposons d'une base de données (MySQL) hébergée gratuitement sur les serveurs de ce service, contenant toutes les informations sur les lieux répertoriés sur notre application. À l'aide de plusieurs scripts php eux aussi hébergés sur les serveurs d'AlwaysData, l'application récupère ces données au format JSON via des requettes, et traite ces données pour les afficher sur la carte.

Pour l'ajout de lieux sur la carte par l'utilisateur lui-même, le fonctionnement est exactement le même mais dans le sens inverse : l'application envoie à la base de données les lieux à ajouter.
